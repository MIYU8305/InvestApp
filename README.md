# InvestApp 📈
> **Advanced Asset Management & Investment Analytics Dashboard**

**[✨ View Live Demo](https://invest-app-one-rust.vercel.app/)**

**InvestApp** is a sophisticated investment tracking platform designed to provide investors with deep insights into their portfolio performance. Built with a focus on data integrity and intuitive visualization, it bridges the gap between complex financial data and actionable investment strategies.

## 🌟 Key Features

* **Unified Portfolio Management**: Consolidate stocks, crypto, and other assets into a single, synchronized dashboard.
* **Dynamic Data Visualization**: Interactive charts for asset allocation, sector exposure, and historical growth trends.
* **Precision Performance Metrics**: Automated calculation of ROI, CAGR, and volatility metrics to evaluate risk-adjusted returns.
* **Web-Based Accessibility**: Fully responsive web application deployed via Vercel for instant access.

---

## 🛠 Technical Stack

* **Framework**: Flutter for Web / Dart
* **Deployment**: Vercel (CI/CD Pipeline)
* **State Management**: Bloc / Provider / GetX
* **Architecture**: MVVM (Model-View-ViewModel) for clean, maintainable code.
